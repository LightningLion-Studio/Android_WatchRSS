## 一、定义与适用范围

1. 本应用是面向可穿戴设备场景的内容聚合与阅读工具，当前主要包括 RSS 订阅与阅读、收藏、稍后再看、分享，以及与 B站、抖音相关的扩展能力。
2. 本协议适用于您通过 WatchRSS 客户端实际使用的功能和服务。
3. 若您通过本应用访问第三方平台、网站、内容源或第三方服务，除本协议外，您还应同时遵守相应第三方平台、网站或服务提供方的用户协议、隐私政策、开放平台规则和其他适用规则。
4. 对于日志上传功能、隐私处理规则等事项，我们还将通过《隐私政策》《日志上传说明》等单独文件向您说明；前述文件与本协议共同构成对本服务的整体约定。

## 二、服务内容

根据当前版本及近期确定启用的功能，本服务主要包括：

1. RSS 地址添加、订阅、解析、频道展示、条目阅读、缓存、收藏、稍后再看等功能；
2. B站相关内容浏览、登录、个性化推荐、播放以及平台侧互动能力；
3. 抖音相关登录与近期确定启用的内容浏览能力；
4. 局域网手机输入 RSS 地址、局域网同步收藏/稍后再看；
5. 系统分享、二维码分享等分享能力；
6. 用户主动触发的日志上传与排障协助。

我们有权根据法律法规、监管要求、产品策略、技术状况、运营安排或第三方平台规则变化，对服务内容进行调整、升级、限制、中止或停止，并将根据具体情况通过适当方式通知您。
